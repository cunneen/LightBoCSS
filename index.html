LightBoCSS for EggMaps.

Pure CSS, accessible, mobile-friendly lightbox that has no JavaScript at all.
Note that it agressively loads all images referenced. If you want a version that performs lazy-loading of large images (using JavaScript), see JSLightBoCSS .

It also degrades gracefully when no CSS (or, of course, JS) is available.
Feel free to use the code, according to the terms of the license.

LightBoCSS was created for EggMaps (a Google Maps app for iOS6. Check it out: http://www.eggmaps.com/ ).

You can browse the HTML, the CSS, download a zip file, or browse the GitHub repository.
