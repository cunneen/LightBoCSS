LightBoCSS for EggMaps.

Pure CSS, mobile-friendly lightbox that has no JavaScript at all. Feel free to use the code, according to the terms of the license.

LightBoCSS was created for EggMaps (a Google Maps app for iOS6. Check it out: http://www.eggmaps.com/ ).

You can download a zip file here, or browse the GitHub repository.
